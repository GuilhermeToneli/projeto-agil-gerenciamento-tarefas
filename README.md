# Projeto Ágil – Gerenciamento de Tarefas

## Objetivo
Simular o desenvolvimento de um sistema ágil utilizando GitHub, Kanban e CI.

## Funcionalidades
- CRUD de tarefas
- Prioridade das tarefas
- Testes automatizados
- GitHub Actions

## Metodologia
Kanban com GitHub Projects.

## Mudança de Escopo
Adição de prioridade nas tarefas para melhor gestão.
