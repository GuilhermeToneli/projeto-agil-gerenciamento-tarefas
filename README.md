# Projeto Ágil – Gerenciamento de Tarefas

## Objetivo
Simular o desenvolvimento de um sistema ágil utilizando GitHub, Kanban e CI.

## Funcionalidades
- CRUD de tarefas
- Prioridade das tarefas
- Testes automatizados
- GitHub Actions

## Metodologia
Kanban com GitHub Projects.

## Mudança de Escopo
Adição de prioridade nas tarefas para melhor gestão.

## Gestão de Mudanças

Durante o desenvolvimento do projeto, foi identificada a necessidade
de incluir o controle de prioridade das tarefas, permitindo melhor
organização do fluxo de trabalho da equipe.

Essa mudança foi incorporada ao escopo sem comprometer o cronograma,
demonstrando a flexibilidade da metodologia ágil e a adaptação contínua
às necessidades do cliente.
